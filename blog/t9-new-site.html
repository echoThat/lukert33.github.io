Injecting JavaScript

November 16, 2014 by Luke

I don't know whether to be proud or ashamed of this: I've spent over an hour trying to make it so that the yellow borders around my nav bar links fade in over 800 milliseconds, instead of instantaneously, when you hover your mouse over them. That 800 milliseconds? That's the difference between using css for my hover events, and using JavaScript.

JavaScript, used in this case with the jQuery library, should enable me to animate the increasing opacity of the yellow border, which is entirely transparent if the mouse is not within it. Here's my code:

$(document).ready(function(){ navLink = $('a.blog-nav-item') navLink.hover( function(){ $(this).css('border').fadeIn(300); }, function(){ $(this).css('border').fadeOut(300); }) }) #=>// Uncaught TypeError: undefined is not a function

The error is arising from the #css('border') call. I'll try to work this out when we look further into JavaScript at DBC.

In the course of researching this, I did learn about easing in JavaScript animations. Easing refers to the acceleration and deceleration of an animation's progress from beginning to end. "Linear" and "swing" are the usual values assigned to it. In linear, the animation moves at the same pace until it is done. Swing easing starts and ends slow, with a relatively heavy acceleration in the middle. It provides a level of naturalism--simulating momentum in your animations. These charts show the difference in progress toward an animation's end state over time:

You can see some really well done demonstrations of animation easing at the jQuery UI site.

So, I'm learning about concepts and failing to execute them well...I think that's supposed to be part of the process.

The goods news is that my fiance and my parents, none of whom know the difference between CSS and JavaScript, like the instantly rendered border currently deployed in the nav bar. Tomorrow, on my first day on site at DBC, I imagine I'll meet many people who will appreciate the huge difference illustrated by those missing 800 milliseconds.
